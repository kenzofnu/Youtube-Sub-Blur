## Name
YouTube Sub Blur

## Short Description (132 chars max)
Blur hard-coded subtitles on YouTube videos with a customizable overlay box. Perfect for language immersion and listening practice.

## Detailed Description

Hide hard-coded (burned-in) subtitles on YouTube videos with a movable, resizable blur box.

Designed for language learners who want to immerse without being distracted by on-screen subtitles — especially useful for Japanese content with hardsubbed translations.

FEATURES:
• Toggle the blur box on/off with a keyboard shortcut (Alt+B by default)
• Drag to reposition and resize to fit any subtitle area
• Box position and size are saved automatically
• Scales correctly between windowed, theater, and fullscreen modes
• Adjustable blur strength (1–40)
• Option to auto-show the blur box on every video
• Customizable hotkey via chrome://extensions/shortcuts

HOW TO USE:
1. Open any YouTube video
2. Press Alt+B to show the blur box
3. Hover over the box to see drag and resize handles
4. Move and resize it to cover the subtitle area
5. Press Alt+B again to hide it

Your box position is remembered so you only need to set it up once.

PERMISSIONS:
• Storage — to save your preferences (blur strength, box position, default on/off)
• No data is collected, transmitted, or shared. Everything stays in your browser.
