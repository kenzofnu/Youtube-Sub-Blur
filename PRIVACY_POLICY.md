# Privacy Policy — YouTube Sub Blur

**Last updated:** March 10, 2026

## Overview

YouTube Sub Blur is a Chrome extension that places a blur overlay on YouTube videos to hide hard-coded subtitles for language immersion.

## Data Collection

This extension does **not** collect, transmit, or share any personal data or browsing activity. There are no analytics, tracking, or advertising services.

## Data Storage

The extension stores the following preferences using Chrome's `storage.sync` API:

- Whether the blur box is shown by default (on/off)
- Blur strength (a number between 1 and 40)
- Box position and size (as ratios relative to the video)

This data stays entirely within your browser. If you are signed into Chrome, these preferences sync across your devices using Google's built-in sync. No third-party servers are involved.

## Permissions

- **Storage**: Used solely to save and retrieve your preferences listed above.
- **Host access (youtube.com)**: Required to inject the blur overlay into the YouTube video player. The extension does not read or collect any page content.

## Contact

For questions or issues, you can reach the developer through the Chrome Web Store support page for this extension.
